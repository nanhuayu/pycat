# PyCat v0.0.1

> Release notes for GitHub Release

## Suggested release title

`PyCat v0.0.1`

## Suggested tag

`v0.0.1`

## Release body

PyCat Agent 首个公开发布版本现已可用。该版本围绕 **LLM chat / agent / tools** 一体化桌面工作流展开，提供桌面优先的 AI 使用体验，并支持以 MCP、技能文件与多模型供应商配置为核心的扩展能力。

## ✨ Highlights

- **Chat / Agent / Tools 一体化**：在同一个桌面应用中统一完成聊天、智能体式任务执行与工具调用。
- **多模型供应商支持**：支持 OpenAI、Claude（Anthropic）、Ollama、Google Gemini、DeepSeek 等多种模型接入方式。
- **MCP / Skills 扩展能力**：支持通过 MCP 接入外部工具和服务，也支持通过技能文件复用能力配置。
- **深度思考渲染**：自动解析并展示 `<think>` / `<analysis>` 内容，适合需要观察模型中间推理展示的场景。
- **桌面端会话体验优化**：支持会话导入导出、消息编辑、分支管理、图片上传、多模态交互与实时性能统计。
- **Windows 独立分发包**：基于 Nuitka 生成 Windows 独立运行目录与压缩包，开箱即可分发。
- **文档与协议完善**：补充中英文 README、截图展示、AGPL-3.0 开源协议与打包脚本。
